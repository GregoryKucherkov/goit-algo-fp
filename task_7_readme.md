Метод Монте-Карло
| Sum   | Probability |  
| ----- | ----------- |
|   2   | 2.79%       |
|   3   | 5.57%       |
|   4   | 8.35%       |
|   5   | 11.18%      |
|   6   | 13.89%      |
|   7   | 16.66%      |
|   8   | 13.82%      |
|   9   | 11.10%      |
|  10   | 8.32%       |
|  11   | 5.54%       |
|  12   | 2.78%       |
| ----- | ----------- |

Ми бачимо,  що точніть при 1000_000 повторень до другого знака після коми, що, дуже точно. Це доказує високу еффективність методу Монте-Карло для розрахунків.
